FreeRTOS Based GCU Project
----------------------------

GSG-701N

By default, the program assigns the following settings to the board:
IP Address: 192.168.1.10
Netmask   : 255.255.255.0
Gateway   : 192.168.1.1
MAC address:  00:18:3E:04:50:84
These settings can be changed in the file initTaskMain.c.
